# MOO_Prob_Matlab

This repository contains MATLAB implementations of multiobjective benchmark
problems used in the numerical experiments reported in the paper:

**"Direct Multisearch – Scale-Invariant (DMS-SI)"**

J. F. A. Madeira

## Contents
- MATLAB implementations of benchmark multiobjective test problems
- Bound specifications and objective evaluations
- Test drivers used in the numerical experiments

## Status
This repository is provided to support reproducibility of the numerical results
reported in the paper. Additional scripts and documentation may be added in
future updates.
